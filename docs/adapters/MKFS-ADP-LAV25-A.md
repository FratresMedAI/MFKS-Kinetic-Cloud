## 3. Amphibious Stow Mode

When entering water, modules stow to ≤ +5° elevation:

| Step | Action |
|------|--------|
| 1 | FCU → SAFE |
| 2 | Release stow pins (2 per module, tool-free) |
| 3 | Fold module to flat position on riser hinge |
| 4 | Re-lock stow pins |
| 5 | FCU → STANDBY (fire disabled until restored) |

Restore time: < 2 min after exiting water.
